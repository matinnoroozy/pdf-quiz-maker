# PDF Question Generator

A simple Streamlit app that creates fill-in-the-blank questions from PDF files.

## How to Install

```
pip install -r requirements.txt
```

## How to Run

```
streamlit run app.py
```

## Features

- Upload PDF  
- Extract text  
- Generate random questions  
- Submit answers and get feedback  
- Save results in `results.csv`  
- Show progress chart

See `requirements.txt`
